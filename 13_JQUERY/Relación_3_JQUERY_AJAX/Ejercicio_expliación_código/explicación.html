Explicación de lo que hace el código: El usuario introduce un número, que puede ser 1, 2 o 3 en el campo del DNI. Después, le da a enviar y se ejecutan las funciones que están vinculadas en el archivo js. Las funciones del archivo js tienen como objeto lanzar una petición al servidor con $getJSON. Es decir, se envia el valor introducido en el campo del DNI del formualario y a cambio recibiremos una respuesta en formato JSON con una serie de datos dependiendo del número de DNI que hayamos introducido y luego se muestran los datos en el div de resultados.
